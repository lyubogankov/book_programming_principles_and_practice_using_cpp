## Notes to self

Basic compilation:
```
g++ -o [exe_name].exe [file].cpp   // if a particular name is desired
g++ [file].cpp                     // if default executable name 'a.out' is ok
```

Compiling with a specific C++ standard:
```
g++ [file].cpp -std=c++11
```
Possible values: `c++11`, `c++14`, `c++17`, `c++20`.
